# 📄 双驱动视光智慧诊疗平台 V1.0 - 产品功能 PRD

## 一、 产品概述

### 产品定位

基于大语言模型（LLM）与 RAG 检索增强技术，面向视光机构与大众的“数据+知识”双驱动高可信视光诊疗与健康管理平台。

### 核心目标

B 端提升基层医生/验光师的诊疗效率与准确率；C 端通过游戏化与智能化手段提升患者依从性与平台粘性。

# 二、 B 端：AI 视光诊疗辅助系统 (Web 端)

B 端系统主要面向视光中心主任、视光医生、验光师及前台管理员，采用 RBAC（基于角色的权限控制）架构。

## 1. Dashboard 页面（数据大屏与 AI 洞察）

### 核心功能

机构运营与医疗质量的全局视图。

### 常规模块

- 核心指标卡：今日就诊人数、初诊/复诊比例、建档总数、各年龄段近视发生率。

- 图表分析：近视防控干预手段（角膜塑形镜、离焦镜、阿托品等）效果对比折线图。

### 🤖 AI 赋能场景 (AI 运营助理)：

- 自然语言查询 (NL2SQL)：医生可通过对话框输入“帮我统计上个月 8-12 岁配戴角膜塑形镜且眼轴增长超过 0.1mm 的患者名单”，AI 直接生成报表。

- 智能周报/月报：AI 自动分析本周诊疗数据，生成《机构视光医疗质量分析报告》，指出异常趋势（如某类并发症增多）并给出管理建议。

## 2. 档案管理（核心业务模块）

### 核心功能

患者全生命周期视觉健康档案的建立、查询与追踪。

### 常规模块

- 患者列表：支持多维度筛选（年龄、屈光度、干预方案、复查时间）。

- 档案详情页：包含基础信息、历次验光数据（电脑验光、主觉验光、眼轴、角膜曲率、眼压等）、处方记录、随访记录。

- 便捷数据录入：支持拍照 OCR 智能识别验光单、角膜地形图报告，自动提取关键数据填入表单。
### 🤖 AI 赋能场景 (CDSS 临床辅助决策)：

- AI 智能诊断建议：在患者档案详情页，点击“AI 辅助分析”，大模型根据患者历次数据（如眼轴变化趋势、屈光度进展），自动生成诊断建议和干预方案推荐（如：“该患者近半年眼轴增长 0.15mm，近视进展较快，建议从单光镜片调整为多点近视离焦镜片，并增加户外活动”）。

- 可信溯源展示：AI 给出的每一条医疗建议，下方必须附带**“知识溯源”**标签（例如：依据《角膜塑形镜验配专家共识2024版》第X章），确保医生敢用、信任。

- 智能随访计划：AI 自动生成个性化的随访时间表，并一键下发至患者的 C 端小程序。

## 3. 组织管理

### 核心功能

支持多院区、连锁视光门店的架构管理。

### 模块详情

- 机构/门店列表管理（新增、停用、基础信息配置）。

- 科室/部门设置（视光科、斜弱视科、干眼门诊等）。

- 数据隔离配置（总院可看全盘数据，分院仅看本院数据）。

## 4. 系统管理

### 核心功能

系统底层支撑与安全审计。

### 模块详情

- 用户管理：医生、账号的创建与停用。

- 角色管理：定义不同角色的默认权限包（如：验光师无权开具处方）。

- 权限管理：细颗粒度的菜单权限、按钮权限、数据权限分配。

- 系统日志：记录所有用户的登录、数据修改、导出等操作，满足医疗合规审计要求。

## 三、 C 端：视光守护小程序 (微信端)

C 端小程序定位为患者（及家长）的“随身眼健康管家”，采用轻量化、游戏化设计。

### 1. 首页：AI 视光辅助问答（核心交互区）

### 核心功能

7x24 小时的专业眼科咨询与导诊。

### 界面设计

类似 ChatGPT 或微信对话界面，顶部有一个可爱的 AI 虚拟形象（如“Eye宝或者瞳灵”）。

### 🤖 AI 赋能场景

- 智能问答：支持语音/文字输入。基于 2666 组真实语料训练，解答家长关于“散瞳是否有害”、“角塑怎么清洗”、“视疲劳怎么缓解”等问题。

- 智能分诊预约：当判断患者描述的症状（如“眼前有黑影飘动”）需要紧急就医时，AI 会自动弹出对应机构的“预约挂号”卡片，引导线下就诊。

- 报告解读：患者拍下在其他医院做的检查单发给 AI，AI 自动提取数据并用大白话解释报告含义。

### 2. 小工具模块（游戏化打卡与促活）

### 核心功能

对标“蚂蚁森林”，通过游戏化机制提升近视防控的依从性（DAU 核心来源）。

### 常规模块

- 护眼能量树/守护宠物：用户的护眼行为可以积攒“护眼能量”，用于树木升级或宠物进化，满级可兑换线下视光中心的免费复查券或周边小礼品。

- 视力自测小游戏：利用手机距离传感器和语音识别，在家进行简单的 E 字表视力粗测。

### 🤖 AI 赋能场景 (智能行为干预)：

- 户外两小时 AI 打卡：结合手机计步和地理位置，或者让用户上传户外活动照片，AI 视觉模型自动识别是否在户外，并发放“能量”。

- AI 眼保健操监督：打开前置摄像头，AI 视觉识别面部穴位，实时语音纠正孩子做眼保健操的姿势和按压力度。

### 3. 个人中心

### 核心功能

个人医疗数据与服务管理。

### 模块详情

- 家庭档案：支持绑定多个就诊人（如大宝、二宝）。

- 我的视觉档案：同步 B 端系统的历次验光数据，以通俗易懂的“折线图”展示视力/眼轴变化趋势。

- 我的预约：查看和管理线下就诊预约。

- AI 护眼周报：每周根据打卡情况和问答记录，AI 自动生成一份《本周护眼总结与下周建议》推送给家长。
